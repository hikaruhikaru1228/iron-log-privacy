IRON LOG

プライバシーポリシー

最終更新日：2026年4月10日

IRON LOG はお客様のデータを外部サーバーへ送信しません。
すべての記録はお使いの端末内にのみ保存されます。

1. 収集する情報

IRON LOG は、アプリの機能を提供するために以下の情報をお客様の端末内に保存します。

  • ワークアウト記録（種目名・重量・回数・セット数・日時・所要時間）
  • 体重記録（体重・日時）
  • アプリ設定（単位・体重・通知設定 など）

これらの情報は端末内のローカルデータベース（SQLite）にのみ保存され、当社または第三者のサーバーへ送信されることはありません。

2. 情報の利用目的

保存された情報は以下の目的のみに使用します。

  • ワークアウト履歴・分析グラフの表示
  • 1RM・消費カロリーの推定計算
  • セット間インターバルタイマーの通知
  • 体重推移グラフの表示
  • ルーティンの保存・呼び出し

3. 第三者への情報提供

当社はお客様の個人情報を第三者に販売・提供・開示することはありません。

IRON LOG が使用しているサードパーティライブラリは以下のとおりです。

  • Expo / React Native — アプリフレームワーク（Meta / Expo Inc.）
  • expo-sqlite — 端末内データベース（外部通信なし）
  • expo-notifications — ローカル通知（外部通信なし）
  • expo-secure-store — 認証情報のセキュアな保存（端末内のみ）

これらのライブラリはいずれも端末外への個人データ送信を行いません。

4. アクセス権限について

IRON LOG はアプリの機能に必要な最小限の権限のみを要求します。

  • 通知（Notifications） — セット間インターバルタイマーの通知に使用

カメラ・位置情報・連絡先・マイクなどの権限は要求しません。

5. データの保存と削除

すべてのデータはお使いの iPhone 内にのみ保存されます。アプリをアンインストールすることで、すべてのデータが端末から削除されます。

アプリ内の「設定」画面から、体重記録・ワークアウト履歴を個別に削除することもできます。

6. 子どものプライバシー

IRON LOG は13歳未満のお子様を対象としていません。13歳未満の方からの個人情報を意図的に収集することはありません。

7. プライバシーポリシーの変更

本ポリシーは必要に応じて更新されることがあります。重要な変更がある場合は、アプリのアップデートを通じてお知らせします。変更後も引き続きアプリをご利用いただいた場合、変更後のポリシーに同意したものとみなします。
